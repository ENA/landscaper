2010/02/11 Landscaper Ver0.0.3.1
このプログラムは想像を絶するほどに無保証です。


libpng,zlib -> zlibライセンス
jpeglib -> 謝辞条項が必要な独自ライセンス
libvorbis,libogg -> 修正BSDライセンス(注：libvorbisの周辺ライブラリは"GPL"であるが、今回は使っていない)
SDL,SDL_mixer,SDL_ttf,SDL_Image,SDL_Net,SDL_gfx,SDL_opengl,OpenGL,GLUT,smpeg -> LGPL(注：SDL_mixerでMAD形式の音を使った場合、必然的に"GPL"となるが、今回は使っていない)
FreeType -> FreeTypeライセンス（BSDライセンスのようなもの)


zlibライセンスにより
・CC2.1のCCBYより、これらがzlibライセンスを用いてることを明記する。
(このプログラムは、zlib,libpngを用いています。このプログラムは次のページからダウンロードできます -> (zlib)http://zlib.net/ -> (libpng)http://www.libpng.org/)
・無保証の表示(上記)
・ソースコードの変更のある場合は、告知文章を削除せず、また追加部分を明記すること。（今回は無い）

修正BSDライセンスにより…(感染性が無いためこれらは無視してよい)
・無保証の表示(上記)
・ライセンス条文の明記

LGPLにより…
・推奨告知の表示(ライブラリではないので不要)
・リバースエンジニアリングを禁止しない(このプログラムは、LGPLの元でライセンスされたライブラリを用いています。よって、リバースエンジニアリングは禁止されません。)
・それぞれのライブラリへのリンクの明記

FreeTypeライセンスにより…
・使用されていることの表示
(このプログラムでは、FreeType2.0ライブラリを内部的に用いています。これは次のサイトで入手することが出来ます -> http://www.freetype.org/)
・ライセンスのReadme.txtを同梱する
